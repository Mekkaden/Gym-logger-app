#  Gym Logger

A high-performance, personal workout logging app built with **React Native (Expo)**. This project is designed for minimal friction during training, focusing on fast data entry and progress tracking.

> **Note:** This is a learning-driven project built for personal use. It focuses on experimentation with React Native, TypeScript, and local data persistence.

---

##  Features

* **Daily Flow**: Automatically loads today‚Äôs workout on launch to get you moving instantly.
* **Fast Gym Input**: Large, accessible input fields for Weight, Reps, and **RIR** (Reps In Reserve).
* **Progress Tracking**: Integrated **1RM Estimation** to monitor strength trends over time.
* **History & Review**:
    * View previous sessions for any specific exercise.
    * **Calendar View** to navigate and review past workouts.
    * Text-based summaries including RIR notes.
* **Offline First**: Uses local persistence for fast, reliable logging without needing a connection.

---

##  Tech Stack

* **Framework**: React Native (Expo Managed Workflow)
* **Language**: TypeScript
* **Storage**: AsyncStorage (Local Persistence)
* **Navigation**: React Navigation
* **UI Components**: React Native Calendars

##  Installation

Ensure you have **Node.js** installed and the **Expo Go** app on your mobile device.

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Mekkaden/Gym-logger-app.git](https://github.com/Mekkaden/Gym-logger-app.git)
    cd Gym-logger-app
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npx expo start
    ```

**To Run:** Scan the QR code in your terminal using the **Expo Go** app (Android) or the **Camera app** (iOS).
